OVERVIEW

The R400NX provides a method of monitoring the position, speed and heading of AIS vessels within VHF range. It can decode of Class A, Class B, Aids to Navigation, SARTS and all other AIS message types. When connected to a PC using the industry standard RJ45 Ethernet connector, the R400NX enables AIS data to be viewed directly, or shared on a local network. The unit can also be mounted at a remote location and AIS data sent via the Internet to a fixed IP address for use on a dedicated server.
